# Rust-PyEnv-Creator

Rust-PyEnv-Creator is a Command Line Interface (CLI) tool written in Rust, designed to simplify the process of setting up Python environments. It allows developers to quickly create isolated Python environments with specified Python versions.

## Features

- Create isolated Python environments.
- Specify the Python version for each environment.
- Cross-platform support (Windows, macOS, Linux).

## Installation

Right now this is in development but once added to path you will be able to use this from the terminal directly.
